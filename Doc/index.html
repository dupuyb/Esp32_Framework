Configuration Features
- Browse files in the Embedded File System
- View the configuration file used at startup
- Upload files to the Embedded File System
- Update ESP32 firmware over the air (OTA)
System Actions
  • - Restart : Initiates an immediate reboot of the ESP32.
  • - Save Config : Saves the configuration to the Embedded File System (EFS). *
  • - Restore : Restores default parameters and removes files **
  • - Reset WiFi : Resets the WiFi configuration. ***

* All parameters in the config.json file will be affected.

** The login, password, and all flags will be reset to their default values. The Embedded File System will be reformatted.

*** The device will reboot in Access Point mode to allow the selection of a new WiFi network.
